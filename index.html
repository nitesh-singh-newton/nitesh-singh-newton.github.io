REST API for an Online Food Application

We have developed this REST API for an online food booking application. This API performs all the fundamental CRUD operations of any online food booking platform, with user validation at every step. This project was developed by a team of four back-end developers during project week at Masai School.

Features
Data Authentication and Validation for all the users.
Owner can login by giving valid credentials.
Owner can add category and provide the item to the respective category.
Customer can login in the application and update their information
Customer can book food item using by adding into cart and destination.
Tech Stack: Java, Spring Framework, Spring Boot, Spring Data JPA, MySQL, Swagger UI, Lambok, Maven
